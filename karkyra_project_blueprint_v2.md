# 🍎 Karkyra: Генеральный план "Apple-дизайна" (v2.0)

Этот документ является «Золотым артефактом» проекта. Он описывает архитектуру и дизайн-систему, которая превратит обычное меню в премиальное веб-приложение в стиле Apple.

## 🎨 1. Визуальный стиль (Aesthetics)

### Цветовая палитра
*   **Background**: Чистый черный (`#000000`) для OLED-эффекта.
*   **Surface**: Темно-серый (`#1C1C1E`) для карточек и элементов управления.
*   **Accent**: Золотой (`#D4A043`) для кнопок "Заказать" и важных акцентов.
*   **Text**: Белоснежный (`#FFFFFF`) для заголовков и мягкий серый (`#8E8E93`) для описаний.

### Типографика
*   Используем шрифты **Google Fonts: Outfit** или **Inter**.
*   Заголовки: Толстые (`Bold`), крупные, с легким межбуквенным расстоянием.

---

## 🏗️ 2. Структура приложения

### Главный экран (Home)
1.  **Video Banner**: Полноэкранное видео с эффектом "параллакса" при скролле.
2.  **Smart Categories**: Горизонтальный список категорий с иконками, который "прилипает" к верху (Sticky Header).
3.  **Bento Grid Layout**: Блюда отображаются в виде сетки разного размера (как в виджетах iOS).

### Детали блюда (Product Sheet)
*   Открывается как "шторка" (Bottom Sheet) снизу.
*   Огромная фотография сверху.
*   Кнопка добавления в корзину с тактильной вибрацией.

### 🍕 2.4. Конструктор "Половинки" (Half & Half)
*   **Визуальный выбор**: Интерактивное разделение пиццы на две части.
*   **Логика**: Выбор разных начинок для каждой стороны.
*   **Цена**: Автоматический расчет (напр. цена более дорогой половины + фиксированный сбор).
*   **UI**: Анимированная смена ингредиентов на изображении пиццы.

## 🛠️ 3. Технические фишки (Premium UX)

*   **Моментальная загрузка**: Отсутствие Service Worker (уже сделано) для всегда свежих данных.
*   **Анимации**: Использование `AnimateDo` или `Animations` для плавного появления блюд.
*   **Корзина (Shared Cart)**: Реальное время (Real-time) через Supabase. Если один гость добавил блюдо, у всех за столом оно появилось.
*   **Фишки CENTER PIZZA**:
    *   **"Pizza Tracker"**: Стильный прогресс-бар приготовления в стиле Dynamic Island (iOS).
    *   **Кастомизация бортиков**: Выбор сырного или хот-дог бортика с премиальными иконками.
    *   **Комбо-конструктор**: Быстрое создание сетов "Пицца + Напиток + Соус".

## 📅 5. Система бронирования (Reservations)

При оформлении бронирования стола гость выбирает один из трех сценариев:

1.  **"Закажем на месте"**:
    *   Классическое бронирование времени и количества человек.
    *   Упор на скорость: уведомление кухни о прибытии гостя.

2.  **"Предзаказ из общего меню"**:
    *   Гость выбирает блюда заранее.
    *   Кухня начинает готовить за X минут до прихода (синхронизация с таймером).

3.  **"Банкетное меню" (Спецпредложения)**:
    *   Выбор готовых сетов для больших компаний.
    *   Фиксированная цена на человека или за стол.
    *   Эксклюзивные позиции, которых нет в обычном меню.
